# Code of Conduct

This project expects respectful, constructive participation.

## Expected Behavior

- Be considerate and professional.
- Focus feedback on the work, not the person.
- Assume good intent, but be clear about technical risks.
- Respect privacy and do not share private credentials, tokens, or user data.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language
- Publishing private information without permission
- Deliberately submitting malicious code or unsafe instructions
- Repeated off-topic or disruptive behavior

## Enforcement

Maintainers may remove comments, close issues, or restrict participation when behavior harms the project or its contributors.
